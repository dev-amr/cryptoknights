The Blockchain

The blockchain is a public ledger of all transactions in the Bitcoin network. Every transaction that has ever happened in Bitcoin is on the blockchain. You can go to the Blockchain.info website and navigate the bitcoin blockchain yourself. If you have an address you can enter it into the website, check the Bitcoin balance and also see the addresses which have transacted with that address. The blockchain only ever increases in size as new transactions are added to the ledger. The ledger is built with complex algorithm encryption, the strength of which is supported by the network of wallet users and miners (computers that are used to find new coins). In other words, the more users bitcoin has the stronger and more secure the network becomes. “As it turns out, the processing power of these computers dedicated to mining for Bitcoins is somewhere between six and eight times greater than all of the top 500 supercomputers combined.” Grant Brunner, ExtremeTech.com May ‘13
